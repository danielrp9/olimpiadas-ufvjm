Informações sobre Cadastro de Atletas

Como funciona o Cadastro de Atletas?

O cadastro de atletas permite que você adicione múltiplos atletas de forma rápida. Fique atento às orientações a seguir:

  • 1
    Atletas cadastrados previamente Para se inscrever nas modalidades é necessário ter os atletas inseridos ao sistema. Garanta que todos os atletas que farão parte das equipes já estejam cadastrados no sistema.
  • 2
    Link de Documento Individual No local de inserir o link de documentos deverá ser anexado um link único referente ao documento daquele respectivo atleta que está sendo lançado. Não utilize o mesmo link para múltiplos atletas.

Inscrições Pendentes

Você tem atletas adicionados ao lote que ainda não foram salvos de forma definitiva. Se você sair agora, esses registros serão perdidos permanentemente. Deseja mesmo sair?
